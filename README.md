# pyshark


## 공부 숙제
pyshark 라이브러리 활용해서 특정 네트워크 인터페이스 실시간 캡쳐후


wireshark 같이 헤더 및 데이터 구분하여 출력해주는 도구 만들기

### 개발 환경
python


### 패킷 헤더 타입 종류
___
• 0800 : IP

• 86DD : IPv6

• 8191 : NetBIOS

• 0600 : Xerox XNS IDP

• 0805 : X.25

• 0806 : ARP

• 0835 : RARP

• 6003 : DEC DECnet Phase Ⅳ

• 8137 : Novell Netware IPX

• 8847 : MPLS

• 8863 : PPPoE Discovery Stage

• 8864 : PPPoE PPP Session Stage
